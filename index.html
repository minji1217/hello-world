1 연구 배경 및 목표
Background & Motivation

기존 LaTeX 논문 작성 워크플로우의 한계

  • \\cite{} 입력 시, 적절한 인용 문헌을 매번 직접 검색하고 선택해야 하는 과정이 반복적이며 비효율적이다.
  • 단순 키워드 검색은 작성 중인 문맥과 연구자의 개인 연구 방향을 동시에 반영하기 어렵다.
  • 논문 작성 흐름을 방해하지 않으면서도 문맥을 이해하는 context-aware citation 추천이 필요하다.

3 사용 흐름
Feature Engineering

사용자는 LaTeX 기반 논문 작성 환경에서 평소처럼 본문을 작성하고, \\cite{} 위치에 커서를 둘 때마다 시스템이 자동으로 인용 후보를 추천한다.

FAISS Stage: 약 300편 후보
Retrieval Stage: 약 150편 후보
1st Ranking: Top‑30
Cross Encoder: Final Top‑10
4 사용 특성 · Feature Engineering
Scoring
  • semantic_score: SPECTER2 cosine similarity (context vs. paper)
  • bib_score: user bibliography soft bias
  • graph_score: citation graph proximity, co‑citation 기반 점수
  • citation_count_log: log(1 + citations)으로 스케일 조정된 인용 수
  • recency_score: 최근 논문에 가산점을 주는 exp decay 기반 점수

최종 1차 점수는 score(p) = α·semantic + β₁·bib + β₂·graph + β₃·log(cite) + β₄·recency 의 형태로 계산하며, 상위 후보에 대해 Cross Encoder를 이용한 2차 reranking을 수행한다.

5 핵심 기여
Contribution
  • LaTeX 기반 환경에서 동작하는 context-aware citation recommendation 시스템 제안
  • 사용자 bibliography soft bias를 활용한 개인화된 인용 추천
  • Semantic retrieval과 citation graph를 결합한 hybrid ranking 전략
  • FAISS ANN 검색 → feature 기반 1차 랭킹 → Cross Encoder 2차 랭킹으로 이어지는 multi-stage ranking 구조 설계
